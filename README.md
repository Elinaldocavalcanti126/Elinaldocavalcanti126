## 🚀 Sobre Mim

Sou desenvolvedor Full Stack com experiência prática construindo aplicações completas tanto em **Python** quanto em **React**, com foco em sistemas com regras de negócio reais — persistência de dados, autenticação, controle de acesso e integração com APIs.

Minha trajetória começou automatizando tarefas repetitivas em Python e evoluiu para sistemas com banco de dados e controle de acesso. Hoje, meu projeto mais completo é o **MedControl**, um sistema de gestão farmacêutica hospitalar construído em **React + Supabase** (PostgreSQL + autenticação), com perfis de acesso, CRUD completo e auditoria — a versão evoluída do sistema hospitalar que antes eu construía em Python com Streamlit e SQLite. Gosto de entender o problema de ponta a ponta: da interface que o usuário vê até a estrutura de dados e a lógica de negócio que sustentam a aplicação.

Esse contato direto com dados me levou a construir interfaces para visualizá-los com **Streamlit**, o que evoluiu para sistemas mais robustos — com **banco de dados, login de usuários e regras de negócio**, como o sistema de gestão hospitalar. O passo mais recente dessa evolução foi reconstruir esse mesmo tipo de sistema com uma stack **Full Stack moderna**: o **MedControl**, feito em **React + Supabase**, com perfis de acesso, CRUD completo e auditoria de ações.
